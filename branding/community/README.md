Community Branding
==================================

STATUS
------

Currently empty. No community assets have been contributed yet.


CONTRIBUTING
------------

If you have created alternative branding for your Futon fork, consider
contributing it here to help other community members.

Submissions should:

- Be original work or properly licensed
- Not resemble the official Futon logo (to avoid confusion)
- Include both SVG (source) and PNG formats
- Be submitted via pull request

USAGE
-----

Assets in this directory are available under CC0 1.0 (Public Domain)
unless otherwise noted in individual files.

Fork maintainers are free to use, modify, and distribute these assets
without restriction.
